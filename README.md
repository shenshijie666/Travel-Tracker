How to Run Travel Tracker

First open terminal.

Go to project folder.

eg: cd ~/Downloads/travel_tracker

Make sure you have Python3 install.

python3 --version

If show version number then is OK.

Check the CSV file also in same folder with travel_tracker.py.

Run the program.

python3 travel_tracker.py

Program will show menu on screen. You can choose option and input number to see travel record or add new travel.

If something error happen, check file name and CSV file location.
